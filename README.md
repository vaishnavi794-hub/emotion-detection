# 😊 Real-Time Emotion Detection System

This project is a Real-Time Emotion Detection System that uses a webcam to detect human emotions like Happy, Sad, Angry, Surprise, and Neutral using Deep Learning and OpenCV.

---

## 🚀 Features

- Real-time face detection using webcam
- Emotion classification using trained deep learning model
- Displays emotion on screen
- Simple and easy to use

---

## 🛠️ Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy

---

## 📁 Project Structure
emotion_detection_webcam/
│
├── app.py # Main file
├── model.hdf5 # Trained model

├── requirements.txt # Dependencies
└── README.md # Project description

## ⚙️ Installation

### Step 1: Clone the repository
it clone https://github.com/vaishnavi794-hub/emotion-detection.git
cd emotion-detection

### Step 2: Create virtual environment
python -m venv venv
venv\Scripts\activate

### Step 3: Install dependencies
pip install -r requirements.txt

---

## ▶️ How to Run
python app.py


- Webcam will open
- Face will be detected
- Emotion will be displayed

Press **Q** to exit.

---

## ⚠️ Important Notes

- Use **Python 3.10**
- Use **NumPy < 2.0**
- Good lighting improves accuracy
- Webcam required
